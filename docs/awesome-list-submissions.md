## Submission Process

For each list:
1. Fork the repository
2. Add the entry in the appropriate section (alphabetical order)
3. Submit a PR with title: "Add Kalibr - adaptive model routing"
4. PR description: "Kalibr provides adaptive model routing for AI agents, learning which LLM models work best per task using Thompson Sampling. It integrates with LangChain, CrewAI, and OpenAI Agents SDK. Published on PyPI as `kalibr` and `langchain-kalibr`."
